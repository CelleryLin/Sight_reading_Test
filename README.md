### 使用方式
請先連接支援 midi 輸出之鍵盤，連上後會自動抓取 midi 訊號

> 提示：當網頁接收到 midi 訊號之後背景會變色喔！

點選 Start 就會直接開始了，在螢幕上會隨機產生一個和弦，請直接彈奏你的 midi 鍵盤，越快越好！
